# Import sur FlutLab.io

1. Cree un nouveau projet sur FlutLab (Flutter App).
2. Menu File -> Import Project -> uploader ce zip OU glisser tous les fichiers.
3. FlutLab generera automatiquement les dossiers android/, ios/, web/.
4. Lancer "Pub Get" puis Run (Web ou Android).

## Garanties FlutLab
- 100% Dart/Flutter (aucun code Java/Swift custom).
- Aucune dependance native exotique : seulement flutter_tts et shared_preferences (officiellement compatibles FlutLab).
- Toutes les donnees sont locales dans assets/data/*.json.
- Aucune API distante, pas de SQLite, pas de Firebase.
